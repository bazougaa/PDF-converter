## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/bazougaa/PDF-converter.git
   cd PDF-converter
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the application using Streamlit:
```bash
streamlit run app.py
```
